*PROJETO ACADÊMICO UTILIZANDO SPRINGBOOT / IA COMO SUPORTE TÉCNICO


🛒 Sistema de Controle de Produtos

Projeto desenvolvido com Spring Boot.

🚀 Funcionalidades
- Cadastro de produtos
- Listagem em tabela
- Exclusão de produtos

Tecnologias
- Java
- Spring Boot
- Thymeleaf
- H2 Database

Como rodar
1. Baixar o projeto
2. Rodar a classe DemoApplication
3. Acessar:
http://localhost:8080/
